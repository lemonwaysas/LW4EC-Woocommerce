== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-gateway-lemonway` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin
